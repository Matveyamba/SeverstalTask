### Запуск
npm install

npm run dev

### Функциональность
Можно создать новую текстовую заметку
Заметка редактируется внутри приложения
Сохранение между сессиями - Данные хранятся в `localStorage`
Seed-заметка - При первом запуске автоматически создаётся стартовая заметка

| Функция                               | Описание                                               |
|---------------------------------------|--------------------------------------------------------|
| Несколько заметок                     | Поддерживается список заметок                          |
| Поиск                                 | Фильтрация заметок по тексту                           |
| Выбор заметки                         | Переключение между заметками                           |
| Удаление                              | Удаление текущей заметки                               |
| Индикатор статуса                     | «Сохранено» / «Есть изменения»                         |
| Предупреждение о несохранённых данных | При попытке уйти из заметки появляется модальное окно  |
| Защита от потери данных               | Предупреждение при закрытии вкладки                    |
| Горячие клавиши                       | `Ctrl/Cmd+N` — новая заметка, `Ctrl/Cmd+S` — сохранить |

---

`notesStorage.js`
- `loadNotes()` — загрузка данных
- `saveNotes()` — сохранение
- `ensureSeed()` — создание seed-заметки

`notesLogic.js`
- создание заметки
- удаление
- обновление
- генерация заголовка
- фильтрация

`useNotes.js`
- управление списком заметок
- сохранение
- подтверждение выхода без сохранения
